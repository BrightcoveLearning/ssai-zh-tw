概述

為了幫助調試服務器端廣告，帶有SSAI的動態投放提供了API端點來跟踪廣告調用。

筆記：
  • 要使用SSAI，您的視頻雲帳戶需要配置為動態投放並啟用了SSAI。請連絡您的客戶經理，開始使用此功能。
  • 該調試接口僅適用於VOD SSAI。
  • 有關限制的列表，請參見SSAI概述文檔。

要使用存儲在Video Cloud中的視頻內容調試服務器端廣告，請按照以下步驟操作：

  1. 查看有關API路徑和授權的常規信息
  2. 跟踪廣告配置
  3. 檢索跟踪詳細信息

一般信息

以下信息與所有SSAI API請求有關。

基本網址

SSAI API的基本URL為：

  https://ssai.api.brightcove.com/v1

帳戶路徑

在任何情況下，都將請求特定的視頻雲帳戶。因此，您將始終添加該詞accounts然後是您的帳戶ID和基本URL：

  https://ssai.api.brightcove.com/v1/accounts/your account id

授權

API請求需要一個Authorization標頭：

  Authorization: Bearer your access token

這access_token是一個臨時的 OAuth2 訪問令牌，必須從Brightcove OAuth服務。有關如何獲取客戶端憑據以及如何使用它們來檢索訪問令牌的詳細信息，請參閱Brightcove OAuth概述。

作業

當您請求客戶憑據時，您將需要指定所需的帳戶訪問或操作類型。以下是SSAI API當前支持的操作的列表：

  • SSAI數據：

    video-cloud/ssai/read
    video-cloud/ssai/all

跟踪廣告配置

要對廣告配置進行跟踪，請按照以下步驟操作：

收集信息

收集以下信息以獲取身體您的API請求：

範圍 類型 描述
account_id 字串 用戶帳號ID
playback_config 物件 字段在配置字段詳細信息 SSAI API文檔中的部分。
title_metadata 物件 告訴系統您的內容多長時間才能生成正確的廣告響應。

例如，下面告訴系統內容是 1 分鐘長。用於插入相應的廣告。 
  "title_metadata": {  "duration": "1m" }
videocloud_metadata 物件 可選的。
僅當您使用引用元數據字段的模板變量時才需要。這些定義在廣告變數 SSAI API文檔中的部分。

要求

為SSAI廣告配置創建跟踪。

方法 崗位
網址 https://ssai.api.brightcove.com/v1/accounts/ {account_id} /ssai_debug_vmap/debug.xml
標頭 授權：持票人access_token（看獲取訪問令牌）
內容類型：application / json
樣品體 
  {
    "playback_config":{
      "name": "config_name",
      "vmap_response_namespace": "config_namespace",
      "account_id": "account_id",
      "ad_config": {
        "enable_ads": true,
        "expected_ad_response": "dfp_ad_rules",
        "disable_server_beacons": false,
        "round_up_cue_points": false,
        "template_url": {
        "template": "template_url"
        }
      },
      "extend_beacon_guard_ttl": false
    },
    "title_metadata":{
      "duration": "39s"
    }
  }

檢索跟踪詳細信息

如上一節所述運行廣告跟踪後，您可以使用會話ID檢索有關跟踪的詳細信息。

這session_id指定緩存會話。每個會話都有自己的長度（基於視頻內容長度）。您可以在上一步中從API調用的響應中獲取ID。

會話ID可以在上一節的VMAP響應中找到。這是一個例子：

  <bc:BrightcoveDebug sessionID="your session id"></bc:BrightcoveDebug>

要求

要通過提供會話ID來檢索廣告呼叫跟踪信息，請使用類似於以下內容的GET請求：

方法 得到
網址 https://ssai.api.brightcove.com/v1/accounts/ {account_id} /ssai_traces/ {session_id} /ad_calls
標頭 授權：持票人access_token（看獲取訪問令牌）
內容類型：application / json
