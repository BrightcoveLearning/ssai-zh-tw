概述

使用 SSAI，廣告會將廣告拼接到伺服器端的內容串流中。使用 VOD + SSAI 時，會提供 VAST 廣告錯誤事件的信標。

廣告錯誤事件可為您的廣告工作流程提供洞察力，讓您主動解決問題。確保您的廣告被放送，有助於最大化廣告收益。

廣告錯誤信標

下表定義透過 SSAI 發出信標的廣告錯誤事件，如 VAST 3.0/4.0 規格所定義。

錯誤代碼 類型 描述
100 ErrorCodeXMLParse

XML 剖析錯誤

  • XSLT 轉換上的錯誤
  • 回應是空的
200 ErrorCodeTrafficking

人口販運錯誤。影片播放器收到的廣告類型並非預期及/或無法顯示。

  • 傳回有效的廣告回應，但找不到線性廣告
  • 傳回非漸進式廣告格式
    • VPAID SWF（當預期不是傳遞時）
    • 無效的視訊 MIME 類型
301 ErrorCodeWrapperRequest

包裝器元素或後續包裝元素中提供的 VAST URI 的超時時間。包括請求錯誤，例如無效的 URI，無法訪問或 URI 請求超時，以及與請求 VAST URI 相關的安全性或其他異常。

  • 包裝器 URI 不可用-4xx，5xx
  • 包裝器在設置的超時閾值內沒有響應
302 ErrorCodeWrapperLimit

已達到包裝器限制，由視頻播放器定義。收到太多包裝器響應，沒有內聯響應。

  • 超過包裝跟隨限制
303 ErrorCodeWrapperNoAds

一個或多個包裝後沒有廣告 VAST 回應。還包含來自後援的空白 VAST 回應數目。

  • 沒有透過包裝函式請求收到線性 (漸進式) 廣告回應
  • 包裝器響應是空的
406 ErrorCodeMissingMezz

夾層是必需的，但沒有提供。未刊登廣告。

  • 嚴格實施 VAST 4.0，並要求傳回 Mezz 廣告。可能是可行的 OTT 設備，以確保質量。
  • 龐大 4 特定
407 ErrorCodeProcessingMezz

夾層是第一次下載。未刊登廣告。

  • 任何時候收到有效的廣告素材，但因處理此事件而被移除，都應該被觸發。
  • 龐大 4 特定
900 ErrorCodeUndefined

未定義的錯誤

  • 發生了廣義錯誤，並沒有被其他錯誤定義
